# Haikyuu!! Team Management System 🏐

This is a Python-based project for the course **SE-2506**. It manages volleyball teams using Object-Oriented Programming (OOP) and analyzes team structures.

## 🚀 Key Features
- **OOP Architecture:** Inheritance-based player roles (Setter, Spiker, Libero).
- **Security:** Custom `@check_access` decorator for admin tasks.
- **Data Analysis:** Statistical calculations using the `math` module.
- **File Management:** Automated reporting exported to `.txt` files.
